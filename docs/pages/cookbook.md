# Cookbook

You can load all the following recipes using `(run-command-load-cookbook)`. You will still need to execute `M-x customize-variable RET run-command-recipes RET` to activate the ones you want to use.

- [package-json](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-package-json.el)
- [cargo](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-cargo.el)
- [pyproject](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-pyproject.el)
- [eldev](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-eldev.el)
- [watchexec](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-watchexec.el)
- [dir-locals](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-dir-locals.el)
- [executables](https://github.com/bard/emacs-run-command/tree/master/cookbook/run-command-recipe-executables.el)
